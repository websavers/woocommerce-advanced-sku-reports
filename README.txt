This plugin enables inventory reports sorted by SKU

== Description ==

This plugin adds a new report under the WooCommerce Stock tab that provides inventory reports and allows
SKU sorting.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Go to WooCommerce > Reports > Stock > All stock by SKU
1. Sort by whichever column you like
